# CVAE-GAN
A CVAE-GAN-Based Approach for Imbalanced Dataset Processing in Intrusion Detection for the Internet of Vehicles

Internet of Vehicles (IoV) serves as the core foundation of intelligent transportation systems, significantly enhancing traffic flow efficiency and overall intelligence levels. But in open wireless communication environments, data transmission is vulnerable to malicious attacks, which can trigger abnormal traffic. Intrusion detection technology is a key measure for ensuring the security of vehicle network communications. However, there is often an imbalance between normal and abnormal traffic in the IoV. Imbalanced datasets can cause classification models to exhibit bias toward normal traffic, thereby compromising the accuracy of intrusion detection. Therefore, this paper proposes a data augmentation model—conditional variational autoencoder-generative adversarial network (CVAE-GAN)—which implements data augmentation based on generative adversarial networks (GAN). First, a core framework integrating conditional encoding with probabilistic generation is adopted to synthesize specified minority class samples, effectively balanced the dataset. Second, a multi-objective optimization function incorporating reconstruction loss, KL divergence loss, adversarial loss, and classification loss is constructed to generate samples with clear class boundaries and reduce inter-class overlap. Finally, a threshold filtering mechanism based on classifier confidence is introduced to filter out noise, further enhancing the quality of generated samples. Evaluate the performance of the CVAE-GAN model using the CAN HCRL OTIDS and Car-Hacking datasets. The experimental results show that the proposed CVAE-GAN enhances attack detection performance compared to the baseline model. The model code is publicly available at https://github.com/Codeuser0717/CVAE-GAN.
